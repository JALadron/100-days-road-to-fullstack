# 100-days-road-to-fullstack
En este repositorio voy a ir subiendo los ejercicios y prácticas como desarrollador fullstack, apesar de lo extenso del temario y tecnologías que abarca este tipo de perfil mi intención no es ser fullstack realmente si no simplemente hacer un reto de 100 días programando aunque sea un poquito, para ir progresando poco a poco.

El camino a seguir será en distintas fases:
1ª Fase Frontend developer:
En esta fase me centraré en las tecnologías básicas de un frontend y haré las primeras aproximaciones a librerias y frameworks llegados el caso.
  a) HTML5
  b) CSS3
  c) BOOTSTRAP
  d) TAILWIND
  e) JAVASCRIPT
  f) JQUERY
  g) REACT
2ª Fase Backend developer:
En esta fase ya empezamos a hacer cosas más interesantes como dar persistencia a nuestros datos, utilzando la estructura MVC la intención es crear pequeñas funcionalidades donde podamos reutilizar código, crear contenido dinámico, hacer consultas, CRUD, etc.
  a) SQL
  a) PHP
  b) Laravel
3ª Fase
En esta fase se trata de hacer algún proyecto que combine todo un poco y profundizar en otros aspectos como el despliegue de la app, usar docker por ejemplo, comunicación con datos estructurados en formato json, consultar APIs, etc

Mi intención más que volverme experto en todo es tener una primer acercamiento a las tecnologías web y a la exploración y experimentación por mi cuenta.
