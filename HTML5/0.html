Acordeón de Comandos Git

Referencia rápida para el flujo de trabajo básico.

Configuración Inicial

git config --global user.name "[nombre]"
Define el nombre que se asociará a tus commits.
git config --global user.email "[email]"
Define el correo electrónico para tus commits.

Gestión de Repositorios

git init
Inicia un nuevo repositorio local en el directorio actual.
git clone [url]
Descarga un proyecto completo desde un host remoto.

El Ciclo de Cambios

  • Estado: git status (Muestra archivos modificados).
  • Seguimiento: git add [archivo] (Prepara el archivo para la foto/commit).
  • Confirmación: git commit -m "[mensaje]" (Guarda la captura de los cambios).

Ramas y Fusión

git branch
Lista todas las ramas locales.
git checkout -b [nombre-rama]
Crea una rama nueva y salta a ella inmediatamente.
git merge [nombre-rama]
Combina la historia de la rama especificada con la actual.

Sincronización Remota

git remote add origin [url]
Conecta el repositorio local a un servidor remoto.
git push -u origin [rama]
Sube los cambios locales y establece la rama de seguimiento.
git pull
Trae y combina los cambios del remoto al directorio de trabajo.
